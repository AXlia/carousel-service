## Development

### Installing Dependencies and Starting Service

Install dependencies:
```sh
npm install
```

Starting Service:
```sh
npm run build
npm start
```
When working with Docker use:
```sh
npm run start-container
```
### Running Jest + Enzyme tests

```sh
npm run test
```

### For Seeding DB with fake data

```sh
npm run seed
```
